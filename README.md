# 100-Ogden-Basic-English · 850词长难句

一个基于 C.K. Ogden 基础英语体系的双语学习网页应用，包含100句精选长难句，帮助学习者掌握850核心词汇的实际应用。

## 🌟 项目特点

- **中英双语对照**：每句均提供英文原句与中文译文
- **智能搜索筛选**：支持按关键词搜索句子、译文或主题
- **主题分类浏览**：8大主题分类，涵盖生活、自然、工作、社会等场景
- **优雅的视觉设计**：采用温暖的复古配色方案，阅读体验舒适
- **响应式布局**：完美适配桌面端与移动端

## 📁 项目结构

```
.
├── ogden_850_sentences.html   # 主页面（HTML + CSS + JavaScript）
└── README.md                  # 项目说明文档
```

## 🚀 快速开始

### 方式一：直接打开

使用浏览器打开 `ogden_850_sentences.html` 文件即可开始使用。

### 方式二：本地服务器（推荐）

```bash
# 使用 Python 启动本地服务器
python -m http.server 8000

# 或使用 Node.js
npx http-server -p 8000
```

然后在浏览器中访问 `http://localhost:8000`

## 🎯 使用方法

### 搜索功能
- 在搜索框中输入关键词，实时过滤句子
- 支持搜索英文原句、中文译文或主题

### 分类筛选
- 点击顶部筛选按钮，按主题分类浏览
- 主题包括：生活场景、自然风光、工作商业、社会人文、情感关系、知识学习、冒险旅途

### 统计信息
- 页面顶部显示词汇分类统计：操作词、通用词、图示词、性质词、反义对

### 返回顶部
- 点击右下角按钮快速返回页面顶部

## 📚 词汇体系

Ogden基础英语包含850个核心词汇，分为以下几类：

| 类别 | 数量 | 说明 |
|------|------|------|
| 操作词 | 100 | 表示动作和状态 |
| 通用词 | 400 | 日常交流常用词 |
| 图示词 | 200 | 具体事物名称 |
| 性质词 | 100 | 描述性质和特征 |
| 反义对 | 50 | 成对的反义词 |
